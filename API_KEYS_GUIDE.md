# 🔑 Get Your Free API Keys — 10 Minutes Total

## 1. NASA FIRMS — Fire Detection ⏱ 3 min
1. Go to: https://firms.modaps.eosdis.nasa.gov/api/
2. Click "Get MAP_KEY"
3. Fill name + email → Submit
4. Key arrives in email instantly
5. Paste in .env as: NASA_FIRMS_KEY=your_key_here

## 2. OpenWeatherMap — Rain/Flood ⏱ 2 min
1. Go to: https://openweathermap.org
2. Sign Up → Verify email
3. Go to: My Profile → API Keys
4. Copy the default key
5. Paste in .env as: OPENWEATHER_KEY=your_key_here

## 3. OpenAQ — Air Quality ⏱ 2 min
1. Go to: https://explore.openaq.org
2. Register → Verify email
3. Dashboard → API Keys → Create
4. Paste in .env as: OPENAQ_KEY=your_key_here

## 4. Mapbox — Map Display ⏱ 2 min
1. Go to: https://mapbox.com
2. Sign Up (free — 50k views/month free)
3. Dashboard shows your default token (starts with pk.)
4. Paste in .env as: MAPBOX_TOKEN=pk.your_token_here
